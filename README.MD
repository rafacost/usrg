# Ultimate Skyblock Resource Generator

<p align="center">
  <img src="https://raw.githubusercontent.com/rafacost/usrg/master/src/main/resources/usrg_logo.png" alt="USRG Logo"/>
</p>

Ultimate Skyblock Resource Generator adds compact generator blocks for skyblock packs. Place a generator, power your progression, and produce common skyblock resources without building large vanilla contraptions for every material.

## Current Release

- Minecraft: `1.20.1`
- Loader: `NeoForge 47.1.79+`
- Java: `17`
- Mod version: `1.20.1-1.0.0`

Downloads are available on:

- [CurseForge](https://www.curseforge.com/minecraft/mc-mods/ultimate-skyblock-generator)
- [GitHub Releases](https://github.com/rafacost/usrg/releases)

## Generators

USRG includes 18 vanilla-resource generators:

- Clay
- Cobblestone
- Dirt
- Dye
- End Stone
- Mycelium
- Glowstone
- Grass
- Gravel
- Ice
- Netherrack
- Obsidian
- Random Ore
- Quartz
- Redstone
- Sand
- Snow
- Soul Sand

USRG also includes 3 optional Ex Nihilo: Sequentia generators:

- Dust
- Crushed End Stone
- Crushed Netherrack

Ex Nihilo: Sequentia must be installed to activate the Dust, Crushed End Stone, and Crushed Netherrack generator recipes.

## Optional Integrations

- [Just Enough Items (JEI)](https://www.curseforge.com/minecraft/mc-mods/jei): recipe viewing.
- [Ex Nihilo: Sequentia](https://www.curseforge.com/minecraft/mc-mods/exnihilosequentia): optional Ex Nihilo resource generators.

## Build

Use Java 17.

```powershell
.\gradlew.bat build
```

The release jar is written to:

```text
build/libs/usrg-1.20.1-1.0.0.jar
```
