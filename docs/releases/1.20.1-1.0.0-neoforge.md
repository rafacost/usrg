# Ultimate Skyblock Resource Generator 1.20.1-1.0.0 NeoForge

## Target

- Minecraft: 1.20.1
- Loader: NeoForge 47.1.79+
- Java: 17

## Download

- CurseForge: https://www.curseforge.com/minecraft/mc-mods/ultimate-skyblock-generator
- GitHub release artifact: `usrg-1.20.1-1.0.0.jar`

## Changes

- Ported the 1.18.1 generator set to Minecraft 1.20.1.
- Switched the build to the NeoForge 1.20.1 toolchain.
- Replaced legacy registry events and object holders with deferred registration during the 1.20.1 port.
- Updated block entities, capabilities, chat components, creative tab registration, and client render setup for the 1.20.1 API.
- Added JEI to the development runtime for recipe verification.
- Added Ex Nihilo: Sequentia and NovaCore to the development runtime for compatibility testing.
- Renamed the Cleanstone Generator display name to Stone Generator.
- Fixed the Stone Generator recipe to use a Stone block in the center slot.
- Kept the first port focused on the existing generator behavior. Tiered generators are not included in this release.
- Ex Nihilo: Sequentia generator recipes remain conditional on the relevant Ex Nihilo items existing.

## Optional Integrations

- JEI is recommended for recipe lookup.
- Ex Nihilo: Sequentia enables Dust, Crushed End Stone, and Crushed Netherrack generator recipes.

## Verification

- `.\gradlew.bat build`
- Client smoke test reached the Minecraft menu with USRG, JEI, NovaCore, and Ex Nihilo loaded.
- JEI showed USRG recipes during manual testing.
- Cobblestone and Ore generators were manually tested in creative mode.
- The built jar contains `META-INF/mods.toml`, which is required by the NeoForge 1.20.1 loader line.
