# Ultimate Skyblock Resource Generator 1.21.1-1.0.0 NeoForge

## Target

- Minecraft: 1.21.1
- Loader: NeoForge 21.1.235
- Java: 21

## Download

- CurseForge: https://www.curseforge.com/minecraft/mc-mods/ultimate-skyblock-generator
- GitHub release artifact: `usrg-1.21.1-1.0.0.jar`

## Changes

- Ported the 1.20.1 NeoForge release to Minecraft 1.21.1.
- Switched the build to ModDevGradle 2.0.141 and Gradle 9.2.1.
- Updated the mod metadata to `META-INF/neoforge.mods.toml`.
- Migrated Forge-era package references to NeoForge 1.21 packages.
- Migrated registries from `RegistryObject` and `ForgeRegistries` to `DeferredHolder`, `DeferredBlock`, `DeferredItem`, and vanilla built-in registries.
- Migrated item handler exposure from overridden capabilities to `RegisterCapabilitiesEvent`.
- Updated block entity NBT save/load methods for the 1.21 registry lookup provider.
- Updated generator tooltips for the 1.21 item tooltip context API.
- Kept Ex Nihilo: Sequentia support optional. Dust, Crushed End Stone, and Crushed Netherrack recipes still require the relevant Ex Nihilo items to exist.

## Optional Integrations

- JEI is included in the development runtime for recipe lookup verification.
- Ex Nihilo: Sequentia remains optional. The current 1.21.1 development runtime does not bundle it because the available 1.21.1 temp-fix files are not reliable Curse Maven dependencies.

## Verification

- `.\gradlew.bat build`
- Client smoke test launched with USRG, NeoForge, and JEI.
- The client reached an integrated world and JEI runtime startup.
- Without Ex Nihilo installed, the Ex Nihilo-themed generator models log missing texture warnings for Ex Nihilo textures. This is expected until the optional dependency is present.
